# Smart Home Energy Advisor Agent

## AI-Powered Personalized Energy Management System

**Platform:** IBM Cloud
**Technology:** IBM watsonx Orchestrate

---

## Overview

The **Smart Home Energy Advisor Agent** is an AI-powered virtual energy consultant that continuously monitors household electricity consumption, analyzes usage patterns, predicts electricity bills, and provides personalized recommendations to optimize energy usage.

The agent leverages a knowledge base containing smart meter data, appliance usage logs, and user preferences to generate actionable insights and energy-saving recommendations. By transforming raw electricity consumption data into meaningful information, the system helps households make informed decisions, reduce energy wastage, and lower electricity costs.

---

## Problem Statement

Most households receive only monthly electricity bills, which provide limited information about how electricity is being consumed. Users often do not know:

* Which appliances consume the most electricity
* Why electricity bills increase unexpectedly
* How daily habits impact energy consumption
* How to optimize appliance usage
* What actions can reduce electricity costs

As a result, electricity is often wasted unknowingly, leading to higher bills and inefficient energy management.

---

## Solution

The **Smart Home Energy Advisor Agent** acts as an intelligent energy advisor that continuously learns from household energy consumption data and provides personalized recommendations to improve energy efficiency.

The agent performs the following functions:

* Monitors household electricity consumption continuously
* Analyzes energy usage patterns and trends
* Identifies high-energy-consuming appliances
* Predicts monthly electricity bills
* Provides personalized energy-saving recommendations
* Generates alerts and proactive suggestions
* Helps users optimize energy usage and reduce costs

## Knowledge Base

The agent uses the following knowledge sources:

### 1. Smart Meter Data

* Timestamp
* Total Energy Consumed (kWh)
* Hourly Consumption
* Daily Consumption
* Peak and Off-Peak Usage

### 2. Appliance Usage Logs

* Appliance Name
* Power Rating (W/kW)
* Usage Duration
* Frequency of Usage

### 3. User Inputs

* Number of Family Members
* Monthly Electricity Budget
* Preferred Comfort Settings
* Working Schedule
* Energy-Saving Preferences

## Example Recommendations

* Increase air-conditioner temperature from 22°C to 25°C.
* Run washing machines during off-peak hours.
* Replace conventional bulbs with LED bulbs.
* Avoid keeping appliances in standby mode.
* Reduce simultaneous usage of multiple high-power appliances.
